Codex CLI 快速入门

OpenAI 推出的命令行 AI 编程助手 · 轻量、快速、终端原生

01

Codex 是什么

· 一句话

Codex CLI 是 OpenAI 发布的终端原生 AI 编程助手，2026年2月开源（MIT协议）。与 Claude Code 类似，直接在命令行中理解项目、写代码、执行命令。

💡

核心定位：轻量、透明、开发者友好——所有操作都在终端里完成，不依赖图形界面。

· 核心特性
⚡

终端原生

不依赖 IDE 或 GUI，纯终端操作

🔓

开源免费

MIT 协议，可自托管

🛡️

沙箱执行

代码在隔离沙箱中运行，安全可控

02

安装步骤

· 快速开始
1
安装 Node.js 18+
codex 依赖 Node.js 运行时
2
安装 Codex
npm install -g @openai/codex
3
设置 API Key
export OPENAI_API_KEY=sk-xxx
4
启动
在项目目录运行 codex
03

基本用法

· 常用命令

进入项目目录后运行 codex，进入交互式对话：

  • "解释这个项目" — Codex 自动读取项目结构并概述
  • "给这个函数加注释" — 精准定位并修改代码
  • "帮我跑测试" — 自动执行并分析测试结果
  • "修复这个 bug" — 定位问题并验证修复
04

Claude Code vs Codex

· 快速对比

Claude Code

桌面端+CLI双模式
Projects / Live Artifacts
MCP 生态
适合复杂工作流和项目管理

Codex CLI

纯终端轻量级
沙箱隔离执行
开源免费
适合快速开发和脚本任务

💡

建议：两者不冲突。Claude Code 做项目管理/H5输出/飞书集成，Codex 做快速编码/脚本任务，互补使用。
