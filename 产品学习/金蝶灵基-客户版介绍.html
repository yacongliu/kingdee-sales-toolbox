常见问题
客户最关心的问题，这里都有答案
灵基能对接我们自建的大模型吗？
目前公有云版本暂不支持对接客户自建模型；未来私有云版本将支持该能力，届时需收取席位费。公有云用户可使用灵基内置的高性能大模型服务。
我们有多个公司账套，灵基怎么处理？
实施时会为您配置默认账套。当涉及多账套或无法确定目标时，AI 会主动询问您指定，确保操作精准、不串数据。
灵基的点数（Token）怎么消耗？
灵基点数是综合折算值。以发票处理为例：视觉模型提取文字（计费）→ 大模型结构化（计费）→ ERP 保存（免费）。简单数据查询不消耗 Token，涉及 AI 推理分析的场景才会计费，按实际调用复杂度决定。
我们现有自研系统，灵基能分析其数据吗？
可以。数据分析场景，通过 MCP 协议封装 API 即可让灵基调用分析。如需在金蝶系统中进行核算记账，则需通过集成平台将数据接入金蝶，确保财务合规与审计闭环。
灵基是否支持多语言？
底层架构支持多语言处理能力。当前产品阶段界面暂未开放语言切换功能，后续版本将逐步提供。
非金蝶的低代码平台（如钉钉宜搭）能接入灵基吗？
目前暂不支持。灵基深度依赖金蝶苍穹平台的元数据结构与 API 规范，以确保生成应用的完整性与可靠性。非标准平台缺乏统一的元数据与接口规范，接入后难以保证业务一致性。
